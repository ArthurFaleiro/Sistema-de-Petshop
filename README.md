#Sistema de Agendamento - Petshop:
Sistema desenvolvido em Java para gerenciamento e agendamento de serviços de Petshop.


#Sobre o Projeto:
Este software foi desenvolvido como Trabalho Final (TF) acadêmico, com o objetivo de facilitar a rotina de um Petshop, permitindo o controle de horários e cadastro de tutores e animais.


#Funcionalidades:
* ** Cadastro de Pets e Tutores:** Registro completo de clientes e seus animais de estimação.
* ** Agendamento de Serviços:** Marcação de horários para Banho, Tosa e Consultas Veterinárias.
* ** Interface Gráfica:** Telas intuitivas desenvolvidas com **Java Swing** (Forms/JFrame).
* ** Consulta de Agenda:** Visualização dos horários ocupados e disponíveis.
* ** Gerenciamento:** Edição e remoção de agendamentos.
